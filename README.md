# centcom-openai-agents

Official CENTCOM connector guides and examples for OpenAI Agents SDK.

## What this repo provides

- Integration playbook for mapping Agents SDK HITL interruptions to CENTCOM approvals.
- Minimal examples for pause -> approve/reject -> resume flows.
- Skill file for AI-assisted implementation.

## Quick Start

```bash
pip install centcom
```

Use the guide below to connect OpenAI Agents SDK approvals to CENTCOM:

- Guide: `docs/openai-agents-sdk-connector.md`
- Skill: `skills/centcom-openai-agents.md`
